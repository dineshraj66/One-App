🆘 TROUBLESHOOTING:
─────────────────────────────────────────────────────────────────

Problem: Dashboard doesn't load
  → Check that all 3 files are in the GitHub repo
  → Verify GitHub Pages is enabled
  → Clear browser cache (Ctrl+F5)

Problem: URLs not working
  → Double-check your GitHub username
  → Make sure app repos are PUBLIC
  → Test URLs in browser first

Problem: Firebase data not showing
  → Check browser console for errors (F12)
  → Verify Firebase config is correct
  → Check Firestore collection names
  → Ensure security rules allow reads

Problem: Service worker issues
  → Open DevTools → Application → Service Workers
  → Unregister all workers
  → Clear all site data
  → Refresh page
